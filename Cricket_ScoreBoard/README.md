# Cricket ScoreBoard üèè

A simple **React front-end app** to keep track of a cricket match score.  

## Features

- Add runs (1, 2, 3, 4, 5, 6)  
- Track wickets (maximum 10 per innings)  
- Track balls and overs (6 balls = 1 over)  
- Display current score, wickets, balls, and overs  

## Technologies Used

- React.js (Functional Components + Hooks: useState, useEffect)  

## How to Run

1. Clone the repository:
      git clone https://github.com/github-saba/React-Projects/Cricket_ScoreBoard.git
2. Navigate to the project folder:
       cd CricketScoreBoard
3. Install dependencies:
      npm install
4. Start the development server:
      npm start  
5. Open your browser and go to http://localhost:3000

Usage

Click the +Run buttons to add runs.
Click the Wicket button to add a wicket.
The app automatically updates balls and overs.
